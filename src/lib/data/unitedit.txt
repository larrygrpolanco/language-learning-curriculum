To add a new section to your unit:

First, choose a descriptive ID for your section (like "comprehension-dialogue" or "grammar-practice-2")
Add the section ID to the sectionOrder array where you want it to appear in the sequence
Create a new entry in the sections object using your chosen ID as the key
Fill in all the required fields:

id: Should match the key you used in the sections object
title: A clear, descriptive title for the section
type: The kind of section (comprehension, drill, etc.)
description: A brief overview of what the section covers
content: The main instructional text
audio: Path to the audio file (if needed)
images: Array of image objects (if needed)
